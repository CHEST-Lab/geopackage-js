This opens the GeoPackage in the browser with no server side component. Be aware that large GeoPackages may not work at this time. Sample GeoPackages can be found at http://www.geopackage.org/#sampledata
Here are some example GeoPackages which can be loaded directly into the tool. Click on one of these links initiates a download into the app and will then show the data. Please be patient if you are on a slower connection.
Natural Earth Rivers (http://ngageoint.github.io/GeoPackage/examples/rivers.gpkg)
Simple Sewers test data set (http://www.geopackage.org/data/simple_sewer_features.gpkg)
Technology Test Data Set (http://www.geopackage.org/data/gdal_sample.gpkg)
